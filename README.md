
# 🏥 Hospital Management SQL Analysis

A SQL analytics project that simulates how hospitals use data to monitor patient care, doctor performance, operational efficiency, and revenue generation.

This project demonstrates real-world SQL skills commonly used by Healthcare Data Analysts, Business Intelligence Analysts, and Hospital Operations teams.

---

## 📌 Project Objectives

The goal of this project is to analyze hospital data and answer key business questions such as:

* Analyze patient admissions
* Monitor doctor performance
* Track appointment trends
*  Measure bed occupancy
* Analyze treatment costs
* Identify readmitted patients
* Calculate average length of stay
* Evaluate hospital performance using KPIs

---

## 📂 Database Schema

The project consists of four tables:

### Patients
Stores patient demographic information.

- Patient ID
- Patient Name
- Gender
- Age
- City
- Registration Date

### Doctors
Stores doctor information.

- Doctor ID
- Doctor Name
- Specialization
- Department

### Appointments
Tracks appointments between patients and doctors.

- Appointment ID
- Patient ID
- Doctor ID
- Appointment Date
- Status
- Consultation Fee

### Admissions
Stores hospitalization records.

- Admission ID
- Patient ID
- Admission Date
- Discharge Date
- Diagnosis
- Treatment Cost
